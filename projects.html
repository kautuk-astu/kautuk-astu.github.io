My Projects

Here are some of the research and technical projects I've worked on.

AeroCluster: Distributed Drone Swarm Management System

Designed and implemented a fault-tolerant, ROS2-based distributed system for coordinating drone swarms. The system features master-drone selection, heartbeat monitoring for health checks, load balancing, and automated crash recovery mechanisms.

Technologies Used: ROS2, Python, Distributed Systems, Drones

GitHub Repo

ChampSim Branch Predictor Evaluation

Evaluated and compared the performance of bimodal, g-share, perceptron, and TAGE branch predictors using the Cloudsuite benchmark suite. Further, a hybrid G-Share + TAGE predictor was implemented under a strict 64KB hardware storage constraint.

Technologies Used: C++, Systems Simulation

GitHub Repo

Optical Character and Format Recognition

Developed a web application to extract tabular data from images while perfectly retaining the original layout and format. This project utilized Convolutional Neural Network (CNN) models built with Keras for the OCR task and Streamlit for the frontend visualization.

Technologies Used: Python, Keras, Streamlit, CNNs
